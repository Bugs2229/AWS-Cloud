1. Navigate to terminal (VS code/Bash) make sure you are in the right folder.
2. git branch <name of new branch>
3. git checkout branch <name of branch you want to change to>
4. vim <name of text file> creating text file
5. Hello world > esc > :wq enter    (editing and saving file)
6. git status (show file staged to be pushed)
7. git add <file> 
8. git commit m- "message"
9. git push --set-upstream origin <file/branch>
